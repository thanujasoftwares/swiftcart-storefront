Customize Style

This template now use SASS CSS pre-processor, all styles stored in style.scss

You can customize bacis variables in style.scss

<!-- Bootstrap mixin required for generate button and styling some vendor prefix -->
@import '../bootstrap-sass/bootstrap/mixins';
<!-- Some helper from bootstrap 4 -->
@import 'helper';
<!-- Robotodraft Font Family -->
@import 'robotodraft';

$font-family-base: 'RobotoDraft','Roboto',sans-serif;
$font-color-base: #666;
$border-radius-base: 0;

$font-color-footer: #bababa;
$background-color-footer: #373939;

There is variable $border-radius-base, to make a flat style set the value to 0
If you want to make a rounded style set $border-radius-base value to higher than 0 (4px for example of this template)

You also can customize color style. The color style stored in _color.scss
All color style used by this template is based from material design color https://material.google.com/style/color.html#color-color-palette 

// Teal
$color-50 : #e0f2f1;
$color-200 : #80cbc4;
$color-400 : #26a69a;
$color-500 : #009688;
$color-600 : #00897b;
$color-700 : #00796b;

Compile New Style

After customizing your style, you need to compile scss file to css file.
Before that, make sure you have installed sass. For more details see http://sass-lang.com/install

Compile your style using this command

sass -t expanded style.scss:new_style.css

For compile minified version use this command

sass -t compressed style.scss:new_style.min.css

Applying New Style

To apply new style, follow step below

  • Find tag <link href="css/style.teal.flat.css" rel="stylesheet" id="theme">
  • Replace style.teal.flat.css with your new style
  • Remove following tags 
    <div class="chooser chooser-hide">
  <div class="chooser-toggle"><button class="btn btn-warning" type="button"><i class="fa fa-paint-brush bigger-130"></i></button></div>
  <div class="chooser-content">
    <label>Color</label>
    <select name="color-chooser" id="color-chooser" class="form-control input-sm selectpicker">
      <option value="indigo">Indigo</option>
      <option value="red">Red</option>
      <option value="teal">Teal</option>
      <option value="brown">Brown</option>
    </select>
    <label class="m-t-1">Style</label>
    <select name="style-chooser" id="style-chooser" class="form-control input-sm selectpicker">
      <option value="flat">Flat</option>
      <option value="rounded">Rounded</option>
    </select>
  </div>
</div>
  • Open mimity.js with your text editor, then remove or comment following script 
    // Change Color Style ======================================================================
$('.chooser-toggle').click(function(){
  $('.chooser').toggleClass('chooser-hide');
});
if (localStorage.getItem('mimity-color') == null) {
  localStorage.setItem('mimity-color', 'teal');
}
if (localStorage.getItem('mimity-style') == null) {
  localStorage.setItem('mimity-style', 'flat');
}
set_color();
$('#color-chooser').change(function(){
  localStorage.setItem('mimity-color', $(this).val());
  set_color();
});
$('#style-chooser').change(function(){
  localStorage.setItem('mimity-style', $(this).val());
  set_color();
});
